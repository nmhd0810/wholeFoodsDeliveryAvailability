# Check Whole Foods Availability
It's an automated script to check the delivery availability in Whole Foods. 

### Instructions:
1) > pip install -r requirements.txt
2) login to Amazon.com in your local Chrome
3) run the script in your local, to generate "cookies_amazon.pkl"
    > python check_availability.py
4) [Optional] deploy the script to a server(without Chrome)
   1. copy "cookies_amazon.pkl" and "check_availability.py" to server
   2. > python check_availability.py
